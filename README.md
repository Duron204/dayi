# 学习项目集合

这是一个包含多个学习项目的仓库，主要用于学习 HTML/CSS 和 Python。

## 项目结构

### HTML/CSS 学习

- `01_welcom` - 入门欢迎页面
- `02_文字与段落` - 文字与段落样式
- `03_图像和超链接` - 图像和超链接使用
- `04_结构标签和分组标签` - HTML 结构标签
- `05_页面交互标签，层次语义标签和全局属性` - 交互与语义标签
- `06_css3基础` - CSS3 基础知识
- `08_id选择器，伪选择器和表格样式` - 选择器与表格
- `010_盒子模型` - CSS 盒子模型
- `011_元素的浮动` - 元素浮动布局
- `012_元素的定位` - 元素定位

### Python 学习

- `python学习` - Python 学习笔记和练习
- `opencv` - OpenCV 图像处理学习

## 使用方法
